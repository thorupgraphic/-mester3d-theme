# Mester3D - Shopify Theme

En moderne, kreativ Shopify theme til 3D-printede produkter.

## 📁 Struktur

```
mester3d-theme/
├── assets/           # CSS og JavaScript
├── config/          # Settings schema
├── layout/          # Hovednayout
├── locales/         # Oversættelser
├── sections/        # Sektioner (header, footer, hero, etc.)
├── snippets/        # Genbrugelige komponenter
└── templates/       # Siden templates
```

## 🎨 Farvepalette

- Primær: `#FFD700` (Gul)
- Sekundær: `#FF8C00` (Orange)
- Accent: `#000000` (Sort)

## 🚀 Import til Shopify

1. Download tema som ZIP
2. Gå til Online Store → Themes
3. Klik "Add theme" → "Upload"
4. Upload ZIP-filen

## ✨ Features

- ✅ Hero banner
- ✅ Produktgrid
- ✅ Responsiv design
- ✅ Dansk sprog
- ✅ Modern animationer
